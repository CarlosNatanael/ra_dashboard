O que significa cada Status?

Estes são os status que um set pode ter durante o processo de revisão:

Pending (Pendente)

Este é o status padrão. O set foi enviado pelo admin e está na fila principal, aguardando um Revisor de Código (CR).

In_Discussion (Em Discussão)

Um CR "assinou" (claimed) o seu set e está ativamente testando. O ícone de desafio (trigger) está sendo testado, a lógica está sendo verificada contra save states, e as descrições estão sendo revisadas. Fique de olho no Discord ou na página do jogo para feedback.

In_Review (Em Revisão Oficial)

O set passou pela revisão inicial (peer review) e agora está com um Revisor de Código (CR) oficial para a aprovação final. Esta é a última etapa antes da promoção.

Approved (Aprovado)

Parabéns! Seu set foi aprovado e movido para a página de Histórico. O próximo passo é verificar os ticket abertos e sempre solicitar revisão quando pedir para promover ou rebaixar alguma conquista.
